¿Qué es Git?

Git es un sistema de control de versiones creado por Linus Torvalds. Te permite registrar la historia de tu proyecto, trabajar en equipo y probar ideas sin miedo a perder tu trabajo.

✨ “Con Git no solo escribes código, escribes historia”.

⚙️ Instalación de Git

🖥️ Windows

Descargar

🐧 Linux

sudo apt install git

🍏 macOS

brew install git

🚀 Comandos Básicos

📂 Inicializar repositorio

git init

➕ Agregar archivos

git add .

💾 Guardar cambios

git commit -m "mensaje"

🌐 Subir a GitHub

git push origin main
